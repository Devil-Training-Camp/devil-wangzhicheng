# Packdep

A tool to help you analyze the dependency relationships in your package.json. It supports lock files from npm, yarn, and
pnpm.

## How to use

1. install
   ```shell
   npm install -g packdep
   ```
2. analyze
   ```shell
   packdep <root of your project>
   ```
   example
   ```shell
   packdep ./packdep
   ```
